Computação e Inteligência Artificial
Redes neurais artificiais (RNA)

Redes neurais ou redes neurais artificiais são sistemas computacionais que tentam simular o comportamento dos neurônios do cérebro humano. Os neurônios se comunicam em redes, transmitindo informações entre eles.

 

“Redes Neurais Artificiais são técnicas computacionais que apresentam um modelo matemático inspirado na estrutura neural de organismos inteligentes e que adquirem conhecimento através da experiência. Uma grande rede neural artificial pode ter centenas ou milhares de unidades de processamento; já o cérebro de um mamífero pode ter muitos bilhões de neurônios.” icmc.usp

 

Um nó representa um neurônio que se comunica com outros através de diferentes camadas, formando uma rede com diversos caminhos possíveis. Cada entrada de dados pode ativar um nó que realiza um processamento e passa o sinal em diante. Essas ativações são baseadas nos dados de entrada com diferentes funções para cada caminho. As ativações dependem dos pesos de cada sinal dos nós encadeados e que aumentam cada vez que são ativados. Os caminhos com maior peso são os mais prováveis.

Isso permite que um sistema como esse possa reconhecer uma imagem sua em uma rede social após milhares de experiências e reforços. O aprendizado profundo utiliza essas informações das redes neurais para decidir qual é a imagem mais provável de ser sua.

 

“A maioria das redes neurais está totalmente conectada de uma camada para outra. Essas conexões são ponderadas; quanto maior o número, maior a influência que uma unidade tem sobre a outra, semelhante a um cérebro humano. Conforme os dados passam por cada unidade, a rede está aprendendo mais sobre os dados. Do outro lado da rede estão as unidades de saída, e é aí que a rede responde aos dados que foram processados.” Disponível em: https://www.oficinadanet.com.br/tecnologia/25007-o-que-sao-as-redes-neurais-artificiais

 

A quantidade de neurônios depende do problema. A rede apresenta sempre uma camada de entrada e outra de saída. A quantidade de camadas internas, camadas ocultas, depende da implementação.

Existem muitas possibilidades de arquitetura que diferem no propósito. Isso traz muitas possibilidades para implementações de projetos com redes neurais.

 

Redes neurais em IA

Os tipos de aprendizagem de máquina com redes neurais também subdividem-se em: aprendizado supervisionado, aprendizado não supervisionado e aprendizado por reforço.

No aprendizado supervisionado é necessário treinar a rede neural com conjunto de dados rotulados disponíveis. Quanto mais informação é passada, melhor os pesos e melhor os resultados. O processo de treinamento vai ajustando os pesos. O ajuste dos pesos é dado pelo processo de backpropagation.

 

Deep Learning

Deep Learning ou aprendizado profundo é um ramo da aprendizagem de máquina (machine learning) que tem como base a utilização de redes neurais artificiais. Nessa abordagem, as máquinas tentam aprender de forma mais parecida com os humanos, utilizando  processamento de informações através de nós que representam neurônios. Esse sistema é possível hoje devido a uma complexa e forte rede de nuvens de dados que disponibiliza um número enorme de informações.

No deep learning a imitação da forma de processamento do cérebro humano é utilizada para reconhecer objetos, vozes, imagens, traduzir idiomas e tomar decisões. No tratamento dos dados o aprendizado profundo das máquinas cria padrões que serão utilizados na tomada de decisão.

Esses muitos dados vindos de diversas fontes demorariam muito para serem tratados um a um com o mesmo procedimento.  O aprendizado profundo utiliza redes neurais para criar níveis de validação desses dados.

A deep learning  (DL) pode ser utilizada ainda para reconhecer uma fraude num cartão de crédito, por exemplo.

O DL utiliza cascatas de diversas camadas de unidades de processamento não linear. Que faz a extração e transformação das características.

A qualidade do conjunto de dados é fundamental para rendimentos satisfatórios.

Para trabalhar com aprendizado profundo você pode começar com as seguintes ferramentas: Keras, Tensor Flow, Theano e CMTK.
