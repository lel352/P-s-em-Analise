Modelagem de dados e aprendizagem estatística
Estatística básica

Ciência utilizada na análise de dados de determinado campo em estudo. A partir dos dados de uma amostra, que representam parte do problema, podemos inferir informações sobre toda população.  Podemos inferir importantes informações sobre uma população a partir da análise de dados de uma amostra dessa população.  Os fundamentos de estatística não dependem do campo que estamos estudando, são sempre os mesmos.

O estudo de probabilidade e estatística nos dá ferramentas para modelar, analisar e definir situações mais difíceis de configurar, conscientes de sua incerteza. No campo empresarial a probabilidade é utilizada em processos no qual a tomada de decisões tem grande quantidade de incerteza.

Conceitos básicos tipos de variáveis, amostra, parâmetro e população.

População é o conjunto de todos os elementos cuja as características estão sendo estudadas.

Conjunto de dados é uma coleção com N quantidades de n-tuplas (lista ordenada de n elementos). N é o conjunto de tuplas de n elementos em cada conjunto de dados. Muitas vezes não conhecemos alguns elementos de algumas tuplas.

Dados são conjuntos de informações observadas de variáveis qualitativas ou quantitativas. Um conjunto de dados tabulados pode corresponder a uma ou mais tabelas com registros de conjunto de dados.

Um conjunto de dados é uma descrição de diferentes instâncias do mesmo fenômeno. Variável é essa característica que pode ter diversas formas, mas devem ser da mesma coisa. A forma mais simples para apresentarmos os resultados de uma coleta de dados é uma tabela. Se a quantidade de dados for muito grande a tabela pode não ser útil, pois fica difícil de ler e comparar os resultados.

Medidas importantes para estatística básica

- Medidas centrais média, mediana e moda.

- medidas de dispersão variância e desvio padrão.

Aprendizagem estatística

Como no machine learning as máquinas aprendem  com dados de experiências passadas, a análise estatística desses dados é fundamental. Não é necessário observar todos os dados, mas os resultados das análises estatísticas dos dados disponíveis permitem a realização de previsões. A quantidade e a qualidade dos dados  e da estatística irá determinar a acurácia das previsões.

O aprendizado de máquinas só é possível existir hoje por conta dos avanços nas tecnologias computacionais e nas teorias e técnicas estatísticas.

Como vimos antes, o aprendizado de máquinas supervisionado pode ser de classificação ou regressão. Este utiliza a regressão que vimos agora e aquele trata de mapear as variáveis de entrada em categorias discretas.

Para aplicação é necessário uma Linguagem de programação. As mais comuns para trabalhos nesse campo hoje são R e Python.

 

Big Data

Vimos que, para um conjunto de dados pequenos, podemos utilizar uma tabela. Para um conjunto muito grande podemos utilizar uma planilha eletrônica. Se o conjunto de dados crescer muito mais, não será possível manipular essas planilhas eletrônicas. Um banco de dados faz-se necessário. Mas, e se os números forem de dimensões muito maiores, como tratar da análise e sistematização desses conjuntos de dados Esse é o trabalho desse campo de estudos que chamamos de Big Data. O conjunto de dados além de grande pode apresentar um grau de complexidade difícil de sistematizar.

Pense na quantidade de dados que passamos a armazenar nas últimas décadas. Números de telefone, placas de carro, e-mail, etc.

“As ferramentas de big data são de grande importância na definição de estratégias de marketing, aumentar a produtividade, reduzir custos e tomar decisões mais inteligentes.” Wikipédia

Esses dados podem crescer exponencialmente a cada momento.

A Modelagem de dados envolve tecnologias para coletar, manipular, analisar, compartilhar, atualizar e exibir dados. Para atender a essas ações em tempo aceitável as bases dos softwares não puderam manter-se com banco de dados relacionados, insuficientes para conjuntos tão grandes.

Os 5 Vs do big data são volume, velocidade, variedade, valor e veracidade.

Os dados estudados podem ser dados estruturados, semi estruturados ou não estruturados.

Data Science

A ciência de dados faz um estudo de dados em diferentes campos de negócios. O cientista de dados vai trabalhar com a captura, visualização, transformação, atualização e análise dos dados. Baseado no tratamento dos dados disponíveis contribuirá para tomadas de decisões futuras inteligentes.

Ciência de dados utiliza métodos científicos, procedimentos, algoritmos e sistemas para extrair conhecimentos e  insights de conjuntos de dados.

Embora tenha relação com aprendizado de máquinas, a ciência de dados é uma subárea desenvolvida da estatística. Muitos dizem que a ciência de dados é a estatística para o meio digital.

Um conceito muito importante em aprendizado de máquinas supervisionado é a regressão.

 

Regressão linear

Na regressão, busca-se um modelo para determinar a relação

entre uma variável aleatória Y que pertence aos números reais e uma tupla x = ( x1 , . . . , xn ) também pertencente aos reais. Ou seja, buscamos uma função que permita determinar valores de Y a partir de uma dada tupla.

Caso a regressão linear não seja boa para ajustar os dados pode-se utilizar uma regressão não linear. Essa é a base do aprendizado de máquinas supervisionado.
